---
title: Clients
description: Integration path for developers using Thumbrella from application code.
---

Thumbrella provides excellent [client libraries](/docs/clients/libraries) for a
variety of languages and environments. These make advanced caching and streaming
integrate directly to language features and the general media libraries for
those platforms.

Rich clients are entirely unnecessary. The Thumbrella 
[http api](/docs/clients/http) is intended to be approachable and usable directly from
the simplest of tools and runtimes.

Higher level web [components](/docs/clients/components) simplify creating interactive and responsive thumbnails for
your applications.
