Thumbrella provides excellent [client libraries](#libraries) for a
variety of languages and environments. These make advanced caching and streaming
integrate directly to language features and the general media libraries for
those platforms.

Rich clients are entirely unnecessary. The Thumbrella 
[http api](#http) is intended to be approachable and usable directly from
the simplest of tools and runtimes.

Higher level web [components](#components) simplify creating interactive and responsive thumbnails for
your applications.
