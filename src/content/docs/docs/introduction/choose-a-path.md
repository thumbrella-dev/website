## Path Summary

- Client path: You want to call an existing service and render thumbnails in your app.
- Server path: You need to run or operate a thumbnail service runtime.
- Service path: You are managing or billing the hosted Thumbrella service.
- Developers path: You are designing or evolving the multi-repo platform itself.
