### Thumbrella Cloud

The <u>free</u> **Thumbrella Cloud** account is <u>recommend for
most users</u>. This include an amazing worldwide edge caching and support all
formats and features of the Thumbrella server. No infrastructure to manage, no
servers to maintain. Provides over 100,000 unique thumbnail renders per month.

Looking for unlimited usage or offline capability? See [self-hosting your own
server](server/) which is free, open-source, and fully featured.

cached and simple results. More complicated renders will result in simpler
placeholders, but continue to work.
See the [cloud limits](/docs/cloud/#limits) page for more details.

Thumbrella also wins by offering a wide range of supported formats and file types.

### Demo Server

The [Thumbrella Demo Gallery](https://demo.thumbrella.dev) has a collection of
media and acts as a <u>free server</u> for the media it hosts. No account
needed, point any client connect string to `https://demo.thumbrella.dev` and
start making requests.
