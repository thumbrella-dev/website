### Thumbrella Cloud

Use the **free** [Thumbrella Cloud](/docs/cloud/) services. They include an
amazing worldwide CDN and support all formats and features of the Thumbrella
server. No infrastructure to manage, no servers to maintain.

If you need unlimited usage or offline capability, see [self-hosting your own
server](/docs/server/) — also free and open source.

There are two measurements of a Thumbrella Cloud account:

- **Cache** — the amount and lifetime of preserved thumbnails in the global,
  distributed edge cache. More cache means more thumbnails served instantly.
- **Render** — the number of thumbnails processed by the server. Cached results
  do not count against render limits.

Each thumbnail is usually between 5 KB and 10 KB in size. Not all thumbnails are
charged at the same rate. See the [cloud limits](/docs/cloud/#limits) page for
more details.


### Self-Hosted

Thumbrella is easy to serve and host on your own. The open source server is
ready to run immediately with a single command. There are **no limits** on
renders, cache size, or usage when self-hosting. See the [server
documentation](/docs/server/) for details.

### AI Platform Servers (coming soon)

Specially deployed Thumbrella servers will be available on hosted platforms
like [fal.ai](https://fal.ai) and [Replicate](https://replicate.com). Run on
these platforms using your existing accounts at no additional cost beyond the
platform's compute pricing. Combine with a [Thumbrella Cloud token](/docs/cloud/#auth-tokens)
for access to the shared global cache.

### Sponsor Edition

Project supporters get access to significantly higher cloud resources and a
custom [Docker](https://docker.com) image with all optional renderers
pre-configured. See the [sponsors page](/docs/sponsor/) for details.

### Demo Server

The [demo gallery](https://demo.thumbrella.dev) has a collection of media
and can be used as a free server for the media it hosts. No account needed —
point any client at `https://demo.thumbrella.dev`.
