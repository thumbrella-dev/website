When the free hosted tier is not enough, upgrade to a managed plan. Your existing integration keeps working — no endpoint changes required.

## What you get

| | Free | Advanced |
|---|---|---|
| Requests / month | 10,000 | Starts at 500,000 |
| Cache regions | Edge CDN | 300+ global PoPs |
| Rate limit | 10 req/s | 500+ req/s |
| Support | Community | Priority email |
| SLA | None | 99.9% uptime |
| Beta access | No | Yes |

## Upgrading

1. Open your [account dashboard](/user/dashboard).
2. Navigate to **Plan** → **Upgrade**.
3. Choose the tier that fits your request volume.
4. Your existing API token keeps working at the new limits immediately.

## Plan details

Plans scale from individual developers running production media pipelines to teams with high throughput requirements. All paid plans include:

- Higher per-request rate limits.
- Priority support with email response SLA.
- Access to beta features before they reach the free tier.
- Dedicated infrastructure options available on request.

Contact [billing@thumbrella.dev](mailto:billing@thumbrella.dev) for volume pricing or custom contracts.
