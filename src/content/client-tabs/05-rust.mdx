The Rust client provides easy, high level functionality for caching, error
control, and other conveniences.

Use `cargo add thumbrella` to get started.

The Rust library provides an async interface built on `reqwests` and
`tokio`.

```rust
let tbr = thumbrella::Client::new(None);
let result = tbr.thumb("https://demo.thumbrella.dev/media/harbor-trucks.mp4").await?;

let img = image::load_from_memory(&result.thumbnail)?;
```
