---
label: JavaScript
---

The Javascript client is written in Typescript and allows running inside
web browsers, Node, Bun, Deno, and other Javascript environments.

Use `npm install @thumbrella/client` to get started (or the equivalent
for your package manager of choice).

The Javascript client provides a rich async interface that supports local
caching and other features.

```js
import { Thumbrella } from '@thumbrella/client';

const tbr = new Thumbrella('<connect string>');
const result = await tbr.thumb('https://demo.thumbrella.dev/media/harbor-trucks.mp4');

img.src = result.thumbnail.as_data_url();
```

The connection string is usually an http server url, or an authentication
token to use with the standard hosted server. It can also be set to the
`$TBR_CONNECT` environment variable to get used automatically.

There are higher level libraries available for `React`, `Astro`, and
other web frameworks. Prefer these component libraries when running in those
types of environments.
